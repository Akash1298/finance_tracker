# Finance-Tracker App

### Steps are necessary to get the application up and running.

- Ruby version: "3.1.3"

- Rails version: "~> 7.0.4", ">= 7.0.4.3"

- Database creation

```
  rails db:create
```

- Database initialization

```
  rails db:migrate
```

- How to run the server

```
  rails server
```
